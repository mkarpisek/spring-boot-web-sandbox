# spring-boot-web-sandbox

## Test
minimal gatling perf test skeleton is in src/test/scala/LoadTest.scala

To execute with maven run for example in 2 shells:

```
mvn spring-boot:run
```

and then tests itself:

```
mvn gatling:test
```
